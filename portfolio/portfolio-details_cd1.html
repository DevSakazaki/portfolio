Recomendador de Filmes

Informações sobre o projeto

  • Categoria: Ciência de Dados, Processamento de Linguagem Natural
  • Data de Término: 06 / Jun / 2023
  • Storytelling do Projeto: Sistema que recomenda filmes de acordo com o filme que o usuário digitar. Nele pode ser feito a escolha da quantidade de filmes a serem recomendados, por enquanto a base de dados é limitada apenas a 5 mil filmes. O sistema é feito com técnicas de PLN como TF-IDF, Similaridade de Cossenos e Distância Euclidiana.
  • Link do Projeto: More Films
  • Técnologias usadas: Python, Streamlit, SkLearn
